# 🕌 بوت الأذكار ومواقيت الصلاة الإسلامي الشامل (ISLAMIC-DZ)

![Telegram Bot](https://shields.io)
![Python](https://shields.io)
![Platform](https://shields.io)

بوت تليجرام إسلامي متكامل ومطور بلغة **Python** مصمم خصيصاً لتقديم مواقيت الصلاة الدقيقة، التقويم الهجري، والتذكير بالأحداث التاريخية الإسلامية البارزة. يتميز البوت بواجهة أزرار مدمجة (ReplyKeyboardMarkup) مريحة وسهلة الاستخدام مباشرة من الهاتف.

---

## ✨ مميزات البوت الرئيسية
* **🕌 مواقيت الصلاة بدقة عالية**: يدعم عرض مواقيت الصلاة لولاية حضرموت بالإضافة إلى 20 ولاية جزائرية مختلفة.
* **📆 التقويم الهجري والميلادي**: عرض التاريخ الحالي بالتقويمين الهجري والميلادي معاً بضغطة زر.
* **📜 أحداث تاريخية إسلامية**: استعراض أبرز الغزوات والأحداث والمناسبات الدينية التي وقعت في نفس اليوم والتاريخ الهجري الحالي.
* **⚙️ تغيير الموقع والولاية**: إمكانية تنقل المستخدم بين الولايات لتحديث المواقيت المخصصة لمدينته تلقائياً وبسهولة.
* **📱 واجهة مدمجة متطورة**: أزرار تفاعلية تظهر مباشرة أسفل حقل الكتابة لتوفير أفضل تجربة مستخدم دون عناء كتابة الأوامر النصية.

---

## 🛠️ المكتبات والتقنيات المستخدمة
تم بناء هذا المشروع وتطويره باستخدام المكتبات التالية:
1. `python-telegram-bot` (الإصدار 21 فما فوق) - للربط والتحكم بالبوت.
2. `requests` - لعمل طلبات برمجية وجلب مواقيت الصلاة من الـ API العالمي الموثوق (Aladhan API).

---

## 🚀 طريقة التشغيل والرفع السحابي (Deployment)

المشروع مهيأ بالكامل للعمل 24 ساعة دون انقطاع عبر منصات الاستضافة السحابية مثل **Railway** أو **Render**:

1. قم بإنشاء متغير بيئة (Environment Variable) في السيرفر باسم:
   * **KEY:** `BOT_TOKEN`
   * **VALUE:** `ضع توكن البوت الخاص بك هنا من BotFather`
2. أمر التشغيل الخاص بالسيرفر (Start Command):
   ```bash
   python main.py
   ```
